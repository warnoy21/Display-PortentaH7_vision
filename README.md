This project shows how to receive JPEG images from H7 over UART and display them on a TFT screen using an ESP32.
![Demo](demo.jpg)

## Features
- UART communication
- JPEG decoding
- TFT screen drawing


## Hardware
- ESP32 Pin 16 & 17 -> Portenta UART 1 
- TFT_eSPI compatible screen
- JPEG source (camera or PC)
